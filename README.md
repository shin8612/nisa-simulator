# 新NISAかんたん積立シミュレーター

Next.jsで作った、ログイン不要・1ページ完結の積立シミュレーターです。

## 必要なもの

- Node.js
- npm

## 動かし方

```bash
npm install
npm run dev
```

ブラウザで http://localhost:3000 を開いてください。

## 公開方法

1. GitHubにこのフォルダをアップロード
2. Vercelで「New Project」
3. GitHubリポジトリを選択
4. Deploy
